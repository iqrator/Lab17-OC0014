# Lab17-OC0014
## 書籍來源：開放文學 

<img alt="alt_text" width="400px" src="OC0014_三教偶拈\OEBPS\cover.jpg" />

### 《三教偶拈》是馮夢龍輯錄的小説集，分別輯錄了儒、釋、道三篇小説，《皇明大儒王陽明先生出身靖亂錄》、《濟顛羅漢淨慈寺顯聖記》和《許真君旌陽宮斬蛟傳》，以體現三教合一的思想。現存於日本東京大學東洋文化研究所。
